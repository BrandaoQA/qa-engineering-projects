# 📊 Análise Qualitativa de Sistema Empresarial - VSSALES

Este repositório contém um relatório de análise qualitativa desenvolvido como parte dos meus estudos em Engenharia de Qualidade de Software.

## 📌 Objetivo
Avaliar a qualidade de um sistema empresarial utilizado no dia a dia, considerando aspectos relacionados à usabilidade, desempenho, organização das informações e experiência do usuário.

## 🏢 Sistema Analisado
- **Nome:** VSSALES  
- **Empresa:** VIMAN Sistemas  
- **Uso:** Sistema empresarial utilizado para gestão de processos internos  
- **Tempo de uso:** Aproximadamente 10 horas diárias  
- **Usuários:** Cerca de 45 colaboradores  

## 🔍 Critérios Avaliados
A análise foi realizada com base nos seguintes pontos:

- Usabilidade
- Organização das informações
- Performance
- Design e interface
- Experiência do usuário (UX)

## 📈 Principais Resultados

### ✅ Pontos Positivos
- Navegação clara e intuitiva  
- Organização eficiente das informações  
- Boa performance durante o uso  
- Interface visual agradável  
- Possibilidade de tema claro/escuro  

### ⚠️ Pontos de Atenção
- Ausência de acesso público ao sistema  
- Falta de documentação ou site oficial acessível  
- Dependência de suporte direto  

## 🛠️ Habilidades aplicadas
- Análise qualitativa de software  
- Avaliação de usabilidade (UX/UI)  
- Observação crítica baseada em uso real  
- Documentação técnica estruturada
